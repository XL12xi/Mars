# Project Summary

Mars Colony Survival is a Java Android game centered on crew management and mission survival.

The current version supports:

- Recruitable crew roles.
- Location-based colony management.
- Simulator training.
- Mission Control threat generation.
- Interactive battles.
- Colony-level crystal rewards.
- Skill upgrades.
- No Death failure handling.

The code demonstrates inheritance, polymorphism, encapsulation, state management, and Android UI integration.
