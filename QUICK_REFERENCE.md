# Quick Reference

Build:

```powershell
.\gradlew.bat assembleDebug
```

Main gameplay path:

1. Recruit or use existing crew.
2. Move crew from Quarters to Simulator for training.
3. Move two crew members to Mission Control.
4. Start mission.
5. Complete battle.
6. Review wins, losses, win rate, and crystals.

Important classes:

- `Colony`
- `CrewMember`
- `MissionControl`
- `InteractiveBattle`
- `LootSystem`
- `SkillUpgradeManager`
