# Improved Project Plan

This plan focuses on making the coursework implementation distinct, complete, and easy to assess.

Core goals:

- Provide a working Android UI instead of only console classes.
- Implement real crew creation, not only predefined sample crew.
- Make Quarters, Simulator, and Mission Control behave as separate locations.
- Generate threats from game state and scale them by completed mission count.
- Include clear failure handling through No Death recovery.
- Track player-facing statistics such as missions, wins, losses, win rate, and medbay visits.

Optional polish:

- Additional visual polish for the existing crew image assets.
- Persistent save/load.
- More test cases and screenshots.
