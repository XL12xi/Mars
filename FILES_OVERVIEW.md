# Files Overview

Important directories:

- `app/src/main`: Android manifest, app resources, and layouts.
- `src/main/java/com/mars/colony/model`: crew and threat models.
- `src/main/java/com/mars/colony/ability`: special ability implementations.
- `src/main/java/com/mars/colony/game`: colony state, mission control, battle, storage, and rewards.
- `src/main/java/com/mars/colony/ui`: Android activities and adapters.
- `src/main/java/com/mars/colony/upgrade`: skill upgrade logic.

Generated directories such as `app/build`, `.gradle`, `.idea`, and `bin` should not be used as source of truth.
