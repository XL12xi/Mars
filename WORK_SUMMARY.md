## 🎮 游戏集成点

### 原始流程

```
招募 → 训练 → 战斗 → (胜利/失败) → 返回
```

### 升级系统集成后

```
招募 → 训练 → 战斗 → 掉落晶体 → 升级技能 → 技能更强 → 战斗更有利 → 返回
```

### 关键装接点

1. **战斗后** - MissionControl.finalizeMission()
   - 计算掉落晶体
   - 分配给存活队员

2. **升级中心** - TrainingCenterActivity (待开发)
   - 显示升级进度
   - 提供升级界面
   - 处理升级请求

3. **数据保存** - SharedPreferences (待开发)
   - 保存升级状态
   - 恢复升级数据
