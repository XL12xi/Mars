# Upgrade System Guide

To add a new ability:

1. Implement `SpecialAbility`.
2. Add the ability to the matching crew during recruitment or default setup.
3. Add a display name in the upgrade UI if needed.
4. Update tests or demo checks.

To upgrade a skill:

1. Earn crystals from successful missions.
2. Open the Training Center.
3. Select the crew and skill.
4. Spend the required crystals.

The current code keeps each ability level from 1 to 5.
