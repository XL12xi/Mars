# Android Studio Guide

1. Open `D:\XueXi\JAVA\project\Mars_remote`.
2. Wait for Gradle sync to finish.
3. Confirm `local.properties` points to a valid Android SDK.
4. Select the `app` run configuration.
5. Start an emulator or connect a physical Android device.
6. Click Run.

If sync fails, check:

- The SDK path in `local.properties`.
- The JDK used by Android Studio.
- Whether the Gradle wrapper files exist under `gradle/wrapper`.
